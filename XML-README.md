### 4. **rss.xml** - RSS Feed

- **Fungsi**: Memberikan update konten kepada subscriber
- **Isi**: Artikel-edukasi, tips, dan event terbaru
- **Fitur**: Full content dengan gambar dan media
- **URL**: `https://litpalm.id/rss.xml`

### 5. **products.xml** - Katalog Produk

- **Fungsi**: Data terstruktur produk dari limbah sawit
- **Isi**: Bio-briket, pupuk organik, arang aktif, matras, kemasan ramah lingkungan, biogas

- **URL**: `https://litpalm.id/education.xml`

### 7. **contact.xml** - Informasi Kontak

- **Fungsi**: Data lengkap kontak dan layanan
- **Isi**: Alamat, telepon, email, jam operasional
- **Fitur**: Tim, lokasi, dan FAQ
- **URL**: `https://litpalm.id/contact.xml`

### 8. **products.xsd** - XML Schema

- **Fungsi**: Validasi struktur XML products
- **Isi**: Definisi tipe data dan aturan validasi
- **Fitur**: Memastikan konsistensi data
- **URL**: `https://litpalm.id/products.xsd`

### 9. **robots.txt** - Robot Instructions

- **Fungsi**: Mengatur crawling behavior mesin pencari
- **Isi**: Allow/disallow rules dan sitemap locations
- **Fitur**: SEO optimization dan crawling control
- **URL**: `https://litpalm.id/robots.txt`

### 10. **xml-sitemap.xml** - Sitemap untuk File XML

- **Fungsi**: Memberikan informasi struktur file XML kepada mesin pencari
- **Isi**: Link ke semua file XML terstruktur dengan prioritas
- **Fitur**: SEO untuk structured data XML
- **URL**: `https://litpalm.id/xml-sitemap.xml`
